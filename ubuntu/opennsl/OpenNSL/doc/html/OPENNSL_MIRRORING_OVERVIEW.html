Port Mirroring APIs

Port mirroring feature allows port traffic to be duplicated on another port (the "mirror-to" port) for debugging or sniffing purposes. The OpenNSL mirroring module allows network application developers to manage mirroring configuration.

OpenNSL Mirroring modes are listed below:

  • OPENNSL_MIRROR_DISABLE Disable Mirroring
  • OPENNSL_MIRROR_L2 L2 Mirroring
